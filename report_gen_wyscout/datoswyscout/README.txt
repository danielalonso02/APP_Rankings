En esta carpeta están los valores asignados a cada variable, según la importancia que se les da en cada una de las posiciones analizadas.

Estos archivos han sido modificados previamente a la reunión celebrada el 10 de julio de 2025. Cuando se disponga de las nuevas modificaciones, se procederá a su implementación.

Los archivos se nombran de la siguiente forma: variables_wyscout_POSICION.xlsx

Las posiciones que es van a evaluar son:

* Portero
* Defensa (central)
* Lateral
* Centrocampista defensivo
* Centrocampista ofensivo
* Extremo
* Delantero Centro

Además se tiene el archivo wyscout_positions_porcentajes.xlsx que tiene la ponderación de las variables ofensivas-defensivas para generar la puntuación táctica global. Este archivo sirve como base para equilibrar la contribución ofensiva y defensiva de cada posición en la evaluación final.
